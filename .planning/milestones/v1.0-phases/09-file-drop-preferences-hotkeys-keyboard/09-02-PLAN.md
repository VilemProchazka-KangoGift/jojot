  <action>
Create a static HotkeyService class in JoJot/Services/ following the project's static service pattern.

**P/Invoke declarations:**
```csharp
using System.Runtime.InteropServices;
using System.Windows.Interop;

[DllImport("user32.dll", SetLastError = true)]
private static extern bool RegisterHotKey(IntPtr hWnd, int id, uint fsModifiers, uint vk);

[DllImport("user32.dll", SetLastError = true)]
private static extern bool UnregisterHotKey(IntPtr hWnd, int id);
```

**Constants:**
- `WM_HOTKEY = 0x0312`
- `HOTKEY_ID = 0x9001` (unique app identifier)
- `MOD_WIN = 0x0008`, `MOD_SHIFT = 0x0004`, `MOD_CONTROL = 0x0002`, `MOD_ALT = 0x0001`, `MOD_NOREPEAT = 0x4000`
- Default: `_modifiers = MOD_WIN | MOD_SHIFT | MOD_NOREPEAT`, `_vk = 0x4E` (VK_N)

**State:**
- `_hwnd: IntPtr` — registered window handle
- `_source: HwndSource?` — for message hook
- `_onHotkeyPressed: Action?` — callback
- `_modifiers: uint` — current modifier mask
- `_vk: uint` — current virtual key code
- `_isRegistered: bool`

**Public methods:**

1. `InitializeAsync(Window window, Action onHotkeyPressed) -> Task<bool>`:
   - Store callback
   - Get HWND via `new WindowInteropHelper(window).Handle`
   - Get HwndSource via `HwndSource.FromHwnd(_hwnd)`
   - Add message hook via `_source.AddHook(WndProc)`
   - Load saved hotkey from database: `await DatabaseService.GetPreferenceAsync("hotkey_modifiers")` and `"hotkey_vk"` — parse as uint, fallback to defaults
   - Call RegisterHotKey — if fails, log warning with `Marshal.GetLastWin32Error()` and return false
   - Set `_isRegistered = true`, return true

2. `UpdateHotkeyAsync(uint modifiers, uint vk) -> Task<bool>`:
   - Unregister current hotkey if registered
   - Register new hotkey with new modifiers + vk (add MOD_NOREPEAT)
   - If success: save to database via SetPreferenceAsync for both "hotkey_modifiers" and "hotkey_vk"
   - Update _modifiers and _vk
   - Return success/failure

3. `GetCurrentHotkey() -> (uint modifiers, uint vk)`: Returns current modifier + vk

4. `GetHotkeyDisplayString() -> string`: Formats current hotkey as readable string (e.g., "Win + Shift + N"):
   - Check each modifier flag and append names
   - Convert vk to key name using `KeyInterop.KeyFromVirtualKey((int)vk).ToString()`

5. `Shutdown()`:
   - Call UnregisterHotKey if registered
   - Remove message hook from HwndSource
   - Clear state

**Private WndProc:**
```csharp
private static IntPtr WndProc(IntPtr hwnd, int msg, IntPtr wParam, IntPtr lParam, ref bool handled)
{
    if (msg == WM_HOTKEY && wParam.ToInt32() == HOTKEY_ID)
    {
        _onHotkeyPressed?.Invoke();
        handled = true;
    }
    return IntPtr.Zero;
}
```

IMPORTANT: Use `System.Windows.Input.KeyInterop` for key conversion (WPF namespace, not WinForms).

**App.xaml.cs changes:**

1. In `OnAppStartup`, after window creation (step 9), add:
   ```csharp
   // ── Step 9.2: Initialize global hotkey (KEYS-01) ──
   bool hotkeyOk = await HotkeyService.InitializeAsync(mainWindow, () =>
   {
       Dispatcher.InvokeAsync(() =>
       {
           string desktopGuid = VirtualDesktopService.CurrentDesktopGuid;
           if (_windows.TryGetValue(desktopGuid, out var existingWindow))
           {
               // Toggle: if foreground and not minimized, minimize; otherwise activate
               if (existingWindow.IsActive && existingWindow.WindowState != System.Windows.WindowState.Minimized)
               {
                   existingWindow.WindowState = System.Windows.WindowState.Minimized;
               }
               else
               {
                   existingWindow.WindowState = System.Windows.WindowState.Normal;
                   WindowActivationHelper.ActivateWindow(existingWindow);
               }
           }
           else
           {
               _ = CreateWindowForDesktop(desktopGuid);
           }
       });
   });
   if (!hotkeyOk)
   {
       LogService.Warn("Global hotkey registration failed — another app may be using the same key combination");
       // Show toast on main window after a brief delay (window needs to be fully loaded)
       mainWindow.Dispatcher.BeginInvoke(System.Windows.Threading.DispatcherPriority.Loaded, () =>
       {
           mainWindow.ShowHotkeyConflictToast();
       });
   }
   ```
