Fix two drag-and-drop bugs:

**Bug 1 — Invisible ghost adorner (line ~1365-1379):**
The VisualBrush in DragAdorner is a live reference to the source element. Setting `_dragItem.Opacity = 0` at line 1369 BEFORE creating the adorner at line 1375 means the VisualBrush paints an invisible element.

Fix: Swap the order. Create the DragAdorner FIRST (while the item is still fully opaque), THEN set `_dragItem.Opacity = 0`. The VisualBrush is live, so we also need to change DragAdorner to use a RenderTargetBitmap snapshot instead of VisualBrush.

Replace the DragAdorner class (line ~3791-3817) to use RenderTargetBitmap:

```csharp
private class DragAdorner : System.Windows.Documents.Adorner
{
    private readonly ImageBrush _brush;
    private readonly double _width;
    private readonly double _height;
    private System.Windows.Point _offset;

    public DragAdorner(UIElement adornedElement, UIElement dragSource, double width, double height)
        : base(adornedElement)
    {
        // Snapshot the element before it becomes invisible
        int pixelWidth = Math.Max(1, (int)Math.Ceiling(width));
        int pixelHeight = Math.Max(1, (int)Math.Ceiling(height));
        var dpi = System.Windows.Media.VisualTreeHelper.GetDpi(dragSource);
        var rtb = new System.Windows.Media.Imaging.RenderTargetBitmap(
            (int)(pixelWidth * dpi.DpiScaleX),
            (int)(pixelHeight * dpi.DpiScaleY),
            dpi.PixelsPerInchX, dpi.PixelsPerInchY,
            System.Windows.Media.PixelFormats.Pbgra32);
        rtb.Render(dragSource);
        rtb.Freeze();
        _brush = new ImageBrush(rtb) { Opacity = 0.5 };
        _width = width;
        _height = height;
        IsHitTestVisible = false;
    }

    public void UpdatePosition(System.Windows.Point position)
    {
        _offset = new System.Windows.Point(position.X + 8, position.Y - _height / 2);
        InvalidateVisual();
    }

    protected override void OnRender(DrawingContext dc)
    {
        dc.DrawRectangle(_brush, null, new Rect(_offset, new System.Windows.Size(_width, _height)));
    }
}
```

Then in the drag initiation code (~line 1365-1379), create the adorner FIRST while item is visible, THEN set opacity to 0:

**Bug 2 — Separator items break indicator rendering (line ~1456-1475):**
In UpdateDropIndicator, when `_dragInsertIndex` points to a separator ListBoxItem, the cast `targetItem.Content is Border border` fails silently (Content is Separator, not Border). The indicator is not shown, but `_dragInsertIndex` remains set, so the drop executes at a position with no visual indicator.

Fix: After computing `_dragInsertIndex`, if the target ListBoxItem's Content is not a Border, scan forward to the next ListBoxItem with Border content. If at end-of-list or all remaining are separators, scan backward instead. Replace lines ~1456-1475 with:

```csharp
// Show horizontal-only line at the drop target position
int indicatorIndex = _dragInsertIndex;

// If target is a separator, find the nearest non-separator item
if (indicatorIndex < TabList.Items.Count)
{
    if (TabList.Items[indicatorIndex] is ListBoxItem targetItem && targetItem.Content is Border border)
    {
        border.BorderThickness = new Thickness(0, 2, 0, 0);
        border.BorderBrush = GetBrush("c-accent");
        _dropIndicatorBorder = border;
    }
    else
    {
        // Separator or non-Border: look forward for next real tab item
        for (int j = indicatorIndex + 1; j < TabList.Items.Count; j++)
        {
            if (TabList.Items[j] is ListBoxItem nextItem && nextItem.Content is Border nextBorder)
            {
                border = nextBorder;
                border.BorderThickness = new Thickness(0, 2, 0, 0);
                border.BorderBrush = GetBrush("c-accent");
                _dropIndicatorBorder = border;
                break;
            }
        }
    }
}
else if (lastSameZoneIndex >= 0)
{
    // Inserting after the last item -- show bottom border on the last same-zone item
    if (TabList.Items[lastSameZoneIndex] is ListBoxItem lastItem && lastItem.Content is Border border)
    {
        border.BorderThickness = new Thickness(0, 0, 0, 2);
        border.BorderBrush = GetBrush("c-accent");
        _dropIndicatorBorder = border;
    }
}
```

Note: The `border` variable in the `else` block needs to be a fresh variable (rename to avoid scope conflict with the outer block). Use proper scoping.
