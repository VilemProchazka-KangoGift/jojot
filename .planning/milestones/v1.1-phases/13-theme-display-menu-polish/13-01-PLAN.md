This task has two parts: (A) convert all font size displays from "Xpt" to "X%", and (B) make tab labels scale with font size changes.

**Part A — Percentage display:**

Add a helper method to the class (near `SetFontSizeAsync`):
```csharp
private static string FontSizeToPercent(int size) => $"{Math.Round(size * 100.0 / 13)}%";
```
Baseline: 13pt = 100%. This gives clean values: 8pt=62%, 10pt=77%, 12pt=92%, 13pt=100%, 14pt=108%, 16pt=123%, 20pt=154%, 26pt=200%, 32pt=246%.

Replace all four "{size}pt" format strings with the helper:
1. Line 2817 (`InitializePreferencesAsync`): `FontSizeDisplay.Text = FontSizeToPercent(_currentFontSize);`
2. Line 2851 (`ShowPreferencesPanel`): `FontSizeDisplay.Text = FontSizeToPercent(_currentFontSize);`
3. Line 2938 (`SetFontSizeAsync`): `FontSizeDisplay.Text = FontSizeToPercent(size);`
4. Line 2945 (`ShowFontSizeTooltip`): `FontSizeTooltipText.Text = FontSizeToPercent(size);`

**Part B — Tab label font scaling:**

In `CreateTabListItem()`:
- Change `labelBlock` FontSize from hard-coded `13` to `_currentFontSize` (~line 295)
- Change `renameBox` FontSize from hard-coded `13` to `_currentFontSize` (~line 312)

In `SetFontSizeAsync()`, after persisting the preference, add a call to `RebuildTabList()` so existing tab items pick up the new font size:
```csharp
private async Task SetFontSizeAsync(int size)
{
    _currentFontSize = size;
    ContentEditor.FontSize = size;
    FontSizeDisplay.Text = FontSizeToPercent(size);
    await DatabaseService.SetPreferenceAsync("font_size", size.ToString());
    ShowFontSizeTooltip(size);
    RebuildTabList();  // Propagate font size to tab labels
}
```

Do NOT scale sidebar date/time text, menu items, or other UI elements — only editor text and tab title labels (per user decision).
