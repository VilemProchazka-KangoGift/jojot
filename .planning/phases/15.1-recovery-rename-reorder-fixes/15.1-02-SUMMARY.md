**Flat row-based recovery panel with per-tab name + content excerpt previews, DockPanel button layout, and +N more overflow indicator**

## Accomplishments
- Added GetNotePreviewsForDesktopAsync to DatabaseService returning (name, excerpt) tuples from SQLite
- Added GetNoteCountForDesktopAsync for total count to drive "+N more" overflow display
- Replaced card-based CreateRecoveryCard (Border with CornerRadius) with flat CreateRecoveryRow (StackPanel)
- Each row displays desktop name (bold), tab count + date (muted), individual tab lines with name + excerpt, and Adopt/Delete buttons
- Both ShowRecoveryPanel and RefreshAfterOrphanAction updated to use new data and layout methods

## Decisions Made
- Existing GetNoteNamesForDesktopAsync preserved alongside new method (may be used elsewhere in codebase)
- CreateRecoveryRow returns FrameworkElement (not Border) since flat layout uses StackPanel container, not card-style Border
- Wrapper StackPanel pattern: row content + optional divider wrapped in outer StackPanel to return as single element; last row skips divider
- DockPanel with LastChildFill=false used for Adopt (left) / Delete (right) button alignment
- Tab name shown with normal weight, excerpt shown as italic em-dash suffix using Inline Runs for mixed styling
