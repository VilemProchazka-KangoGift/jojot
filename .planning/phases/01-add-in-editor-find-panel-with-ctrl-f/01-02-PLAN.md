    **ShowFindPanel(bool showReplace = false):**
    - Call `ViewModel.CloseAllSidePanels()` first (closes preferences/cleanup/recovery AND sets IsFindPanelOpen=false)
    - Set `_findPanelOpen = true`
    - If ContentEditor has a selection (`ContentEditor.SelectionLength > 0`), call `FindReplacePanel.SetFindText(ContentEditor.SelectedText)`
    - Call `FindReplacePanel.Show(showReplace)`
    - Run initial search if find input already has text (query persists across open/close)

    **HideFindPanel():**
    - Set `_findPanelOpen = false`
    - Call `FindReplacePanel.Hide()`
    - Clear match state (`_findMatches.Clear()`, `_currentFindIndex = -1`)
    - Focus ContentEditor

    **WireUpFindPanelEvents() — called from constructor:**
    Subscribe to all FindReplacePanel events:
    - `FindReplacePanel.CloseRequested += (_, _) => HideFindPanel()`
    - `FindReplacePanel.FindTextChanged += OnFindTextChanged`
    - `FindReplacePanel.FindNextRequested += (_, _) => CycleFindMatch(forward: true)`
    - `FindReplacePanel.FindPreviousRequested += (_, _) => CycleFindMatch(forward: false)`
    - `FindReplacePanel.ReplaceRequested += (_, _) => PerformReplace()`
    - `FindReplacePanel.ReplaceAllRequested += (_, _) => PerformReplaceAll()`

    **OnFindTextChanged(object? sender, FindChangedEventArgs e):**
    - If query is empty or _activeTab is null: clear _findMatches, update panel counter, return
    - Call `MainWindowViewModel.FindAllMatches(ContentEditor.Text, e.Query, e.CaseSensitive, e.WholeWord)`
    - Store results in `_findMatches`
    - Set `_currentFindIndex = _findMatches.Count > 0 ? 0 : -1`
    - Call `FindReplacePanel.UpdateMatches(_findMatches, _currentFindIndex)`
    - If matches found, select the first match in ContentEditor (`ContentEditor.Select(pos, queryLen)` and scroll to it)

    **CycleFindMatch(bool forward):**
    - If no matches, return
    - `_currentFindIndex = MainWindowViewModel.CycleIndex(_currentFindIndex, _findMatches.Count, forward)`
    - Select match in ContentEditor and scroll to line
    - Update panel counter via `FindReplacePanel.UpdateMatches(_findMatches, _currentFindIndex)`

    **PerformReplace():**
    - If no matches or _currentFindIndex < 0 or _activeTab is null, return
    - Get replacement text from `FindReplacePanel.GetReplaceText()`
    - Get match position from `_findMatches[_currentFindIndex]`
    - Call `MainWindowViewModel.ReplaceSingle(ContentEditor.Text, matchPos, queryLen, replacement)`
    - Set ContentEditor.Text to new content (this triggers autosave via existing TextChanged handler)
    - Re-run search (OnFindTextChanged equivalent) to refresh matches after replacement
    - Auto-advance: cycle to next match if available

    **PerformReplaceAll():**
    - If no matches or _activeTab is null, return
    - Push undo snapshot BEFORE replacement: `UndoManager.Instance.GetOrCreateStack(_activeTab.Id).PushSnapshot(ContentEditor.Text)`
    - Call `MainWindowViewModel.ReplaceAll(ContentEditor.Text, query, replacement, caseSensitive, wholeWord)`
    - Set ContentEditor.Text to new content
    - Show replacement count feedback: call a method on FindReplacePanel to show "{count} replacements made" briefly (via the ReplaceCountText element, maybe with a DispatcherTimer to auto-clear after 3 seconds)
    - Re-run search to refresh matches (should find 0 if all replaced)

    **Tab switch integration:**

    if (_findPanelOpen && !string.IsNullOrEmpty(FindReplacePanel.GetFindText()))
